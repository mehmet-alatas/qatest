Practice Lab: Built-in User-Facing Locators

Bu sayfa Playwright’in önerdiği locator’ları (getByRole, getByText, getByLabel, getByPlaceholder, getByAltText, getByTitle, getByTestId) uygulamalı öğretmek için tasarlandı. Her bölümde UI üzerinde bir “state” değişimi veya doğrulanabilir çıktı var.

Öneri: Bu dosyayı bir local server ile çalıştır: npx http-server . → http://127.0.0.1:8080/playwright-locators-lab.html

Hedef

Öğrencilerin her locator’u gerçek örnek üzerinde görmesi ve test yazması.

Doğrulamalar

Toast mesajları, sayaçlar, toggle state ve seçili seçenekler ile assert yapılabilir.

İpucu

await expect(locator).toBeVisible() ile stabil test yazmayı vurgula.

getByRole()

Role tabanlı seçim — Button / Checkbox / Link

Accessibility role + name ile en güvenilir seçim.

S1

Uygulama Alanı

Go to footer ↓

Test Case Steps (S1)

  1. Search input’a bir metin yaz (ör: Playwright).
  2. Run Search butonunun enabled olduğunu doğrula.
  3. Enable advanced mode checkbox’ını işaretle.
  4. Run Search butonuna tıkla ve toast mesajını doğrula.
  5. Go to footer link’ine tıklayıp sayfa sonuna kaydığını doğrula.
Önerilen locator örnekleri:
page.getByRole('button', { name: 'Run Search' })
page.getByRole('checkbox', { name: 'Enable advanced mode' })
page.getByRole('link', { name: 'Go to footer' })
getByText()

Metin tabanlı seçim — Dinamik mesaj doğrulama

UI’daki text değişimini assert etmek için ideal.

S2

Uygulama Alanı

Aşağıdaki butona tıklayınca mesaj değişir. Metni yakalayıp doğrulayın.

Status: Idle. Click a button to generate a message.

Test Case Steps (S2)

  1. Generate Message butonuna tıkla.
  2. Sayfada Status: Success metninin göründüğünü doğrula.
  3. Generate Warning butonuna tıkla.
  4. Sayfada Status: Warning metninin göründüğünü doğrula.
Önerilen locator örnekleri:
page.getByText('Status: Success')
page.getByText(/Status: Warning/i)

getByTitle()

Title tabanlı seçim — Tooltip / ikon

title attribute ile ikon/element yakalama.

S6

Uygulama Alanı

Aşağıdaki ikonlar title attribute taşır.

Test Case Steps (S6)

  1. title="help-icon" olan butonu yakala ve tıkla.
  2. Toast mesajında Help geçtiğini doğrula.
  3. title="settings-icon" ve title="info-icon" için aynı doğrulamaları yap.

getByTestId()

TestId tabanlı seçim — Sayaç ve state doğrulama

UI metni değişse bile stabil selector için.

S7

Uygulama Alanı

Counter: 0 Idle

Test Case Steps (S7)

  1. counter-value değerinin başlangıçta 0 olduğunu doğrula.
  2. counter-inc butonuna 2 kez tıkla.
  3. counter-value değerinin 2 olduğunu doğrula.
  4. counter-dec ile 1 azalt, değerin 1 olduğunu doğrula.
  5. counter-reset tıkla, tekrar 0 olduğunu doğrula.
Önerilen locator örnekleri:
page.getByTestId('counter-inc')
page.getByTestId('counter-value')
